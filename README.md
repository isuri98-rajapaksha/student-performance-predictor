# Student Performance Predictor

## 📌 Project Description
This is a machine learning project that predicts student marks based on study hours.

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📊 Features
- Predict marks based on input hours
- Accuracy calculation using R² score
- Graph visualization of dataset
- Trained ML model saved as .pkl file
